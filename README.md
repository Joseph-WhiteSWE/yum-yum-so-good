<h2>install xcode or andriod equivalent</h2>
 <h3>To start:</h3>
 <p>1. npm i</p> 
 <p>2. npm run start</p>
 <p>3. press i </p>
 <em>press r to resart the app at any time</em>

<em>may show red timeout error, that's okay, just wait</em>

 <h3>If fail to start:</h3>
 <p>1. npm start -- --reset-cache</p>
 <p>2. press i</p>
