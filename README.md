 <h3>To start:</h3>

<h4>install xcode</h4>
 
 <div>1. npm i</div> 
 <div>2. npm run start</div>
 <div>3. press i </div>
 <p>4. press r to resart the app at any time</p>

<em>may show red timeout error, that's okay, just wait</em>

 <h3>If fail to start:</h3>
 <div>1. npm start -- --reset-cache</div>
 <div>2. press i</div>
